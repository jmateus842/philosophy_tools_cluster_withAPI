Acerca de El Argumentista

El Argumentista es una aplicacion educativa hecha con Angular. Su proposito es facilitar el aprendizaje de pensamiento critico y filosofia mediante conceptos claros, ejemplos practicos y citas inspiradoras.

Objetivos

  • Brindar definiciones sencillas de conceptos clave.
  • Fomentar la exploracion por categoria y vistas de detalle.
  • Inspirar con frases celebres de filosofos y autores.
  • Consumir la API externa de citas filosoficas.

Tecnologias utilizadas

  • Angular 17 con componentes standalone
  • Bootstrap 5 para estilos responsivos
  • RxJS para manejo reactivo
  • Archivos JSON locales como fuente de datos
  • API externa mock de citas filosoficas
